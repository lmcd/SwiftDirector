LDAPObject

@dynamicMemberLookup
public struct LDAPObject<ObjectClass> : Equatable, Hashable, Identifiable, CustomStringConvertible, CustomDebugStringConvertible where ObjectClass : ObjectClassProtocol

Represents a concrete object of a given object class type. This type can be used to extract attributes for this object.
