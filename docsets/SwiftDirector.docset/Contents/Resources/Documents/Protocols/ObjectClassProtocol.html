ObjectClassProtocol

public protocol ObjectClassProtocol

The base protocol for all object classes. New object classes should always declare a protocol (MyObjectClassProtocol) and a corresponding implementation (MyObjectClass). However, the implementation should alway just be an empty struct just containing the oid and the protocol should have all properties defined in an extension. See e.g. OrganizationalPerson.
